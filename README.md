# Nemopedia

Nemopedia adalah aplikasi mobile berbasis Android yang menyediakan artikel edukatif
berdasarkan kategori tertentu. Aplikasi ini dirancang untuk mendukung pembelajaran
dan pencarian informasi secara praktis melalui perangkat smartphone.
